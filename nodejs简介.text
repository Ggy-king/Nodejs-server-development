# 为什么选择nodejs

1 不同语言都能实现服务端的职责
2 不同语言的区别 {
    语言语法和内置API
    框架
    工具包和软件包(如npm)
}

3 nodejs优势 {
    nodejs使用js语法
    nodejs使用npm
    还需要学习框架和API
}
