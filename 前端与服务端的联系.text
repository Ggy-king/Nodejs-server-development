//前端与服务端

//服务端

{
    1 服务端又称后端 server端
    2 前端是用户可见 可操作的部分 如树枝树叶
    3 服务端为前端提供支撑和营养 如树根
    
}

{
    服务端的职责{
        1 提供前端需要展示的内容
        2 接收前端要提交的数据
        3 储存数据(软件公司很看重数据 会收集各种数据)
    }
}

{
    服务端的表现形式{
        1 前端的ajax
        2 get post
    }
}

{
    前端{
        狭义：网页
        广义：各种客户端 如APP PC客户端等
    }
}
